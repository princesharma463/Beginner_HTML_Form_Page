# Beginner_HTML_Form_Page
Beginner html form page for practice purpose.

🎓 University Registration Form

A simple University Registration Form built using HTML5. This beginner-friendly project demonstrates how to create a structured form using various HTML form elements.

✨ Features

* Personal Details section
* Contact Information section
* Academic Information section
* Gender selection using radio buttons
* Course selection using a dropdown menu
* Extra-curricular interests using checkboxes
* Photo upload option
* Password creation field
* Submit button

🛠️ Technologies Used

* HTML5

📚 HTML Concepts Covered
* Form (<form>)
* Input fields (text, email, tel, date, password, file)
* Labels
* Radio buttons
* Checkboxes
* Textarea
* Select & Option 
* Headings
* Horizontal Rule
* Line Breaks



🎯 Purpose

This project was created to practice the fundamentals of HTML forms and improve understanding of user input elements. It is suitable for beginners who are learning web development.

🚀 Future Improvements

* Add CSS for a modern and responsive design
* Add JavaScript form validation
* Improve accessibility
* Connect the form to a backend database
* Store submitted user information

📷 Preview

Open the index.html file in any web browser to view and interact with the registration form.
